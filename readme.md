# Summary
This is a full stack app for generating invoices/pdfs. There is a form that will allow the user to input line items and other template data for the pdf.

## Tech Stack
### Frontend:
- React.js
- Redux
- Axios

### Backend:
- Node.js
- Express.js
- Nunjucks
- Puppeteer
- MongoDB
- Mongoose
- jsonwebtoken (JWT)
